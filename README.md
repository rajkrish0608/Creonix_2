# Creonix_2

Official playful skeuomorphic & liquid-glass landing page for **Creonix Technologies Pvt. Ltd.** (India's STEM Lab Infrastructure Partner).

## 🚀 Features
- **Skeuomorphic & Liquid Glass Aesthetic:** Curved visual components with backdrop blurs, double-layered claymorphic drop shadows, and inset highlights.
- **Warm Orange-White Gradient Theme:** Beautiful sunset-themed linear gradients matching a clean, visual design.
- **3D Card Hover Tilts:** Interactive kit cards that rotate and tilt based on real-time mouse position.
- **WebGL Orb Backgrounds:** Floating glowing spheres drifting slowly in the canvas.
- **Tactile Active Press Animations:** Physical compression states on all main buttons and inputs.
- **Confetti Form Inquiry:** Built-in form submissions triggering confetti explosions on successfully sent messages.
- **Double Cursor Tracking:** Dual-ring mouse cursor tracker with realistic movement inertia.

## 🌐 Preview Locally
Run the static server to view:
```bash
python3 -m http.server 3000
```
Then visit [http://localhost:3000/](http://localhost:3000/).
